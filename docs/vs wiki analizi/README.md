# VS / Brotato wiki analizi — arşiv klasörü

Bu klasördeki metinler **projenin çok erken döneminde**, Vampire Survivors wiki’sinden ve Brotato arena referansından çıkarılmış **ilham + öncülük** notlarıdır. Ironfall’a birebir kopya değildir; güncel ürün önceliği **`docs/YOL_HARITASI.md`** (özellikle «Referans — VS / Brotato wiki analizi») ve **`docs/YAPILACAKLAR_TOPLU.md`** içinde tutulur.

## Dosyalar

| Dosya | İçerik |
|--------|--------|
| `vs_wiki_analiz_asama1.md` | Harita modifier’lar, karakter/silah/evrim/pasif/pickup, düşman & wave event taslağı, öncelik listesi |
| `vs_wiki_analiz_asama2.md.md` | Arcana, curse, relic, tüccar, power-up / meta fiyat formülü, collection — *(dosya adında çift `.md` uzantısı tarihsel)* |
| `vs_wiki_analiz_asama3.md` | Silah havuzu ekleri, Limit Break, Golden Egg, Coffin, sandık, curse, karakter unlock yolları |
| `brotato_arena_analiz.md.md` | Arena: dalga süreleri, scaling, elite/horde, boss, danger — *(çift `.md` tarihsel)* |
