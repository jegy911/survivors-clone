Codex art (optional second root)
================================
Same layout as `assets/ui/codex_icons/` — resolved **after** `codex_icons` and **before**
tab-specific fallbacks (enemy sprites, map preview, character `codex.png`, etc.).

  res://assets/ui/codex_art/<tab>/<id>.png

`<tab>`: enemy, boss, weapon, item, character, map, glossary (see `codex_icons/README.txt`).

Use this folder for large “design” exports if you want to keep `codex_icons` for small grid icons only.
